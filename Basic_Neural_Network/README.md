약물 복용량에 따른 복용효과에 대한 간단한 표본으로 만들었습니다.
1. 약물을 복용하지 않은 집단(x=0)은 복용효과가 없습니다.(y=0)
2. 약물을 적당량 복용한 집단(x=0.5)은 복용효과가 가장 큽니다.(y=1)
3. 약물을 많이 복용한 집단(x=1)은 복용효과가 없습니다.(y=0)
